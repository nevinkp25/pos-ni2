# **App Name**: Time Slotter

## Core Features:

- Restaurant Selection: Allow users to select a restaurant from a dropdown menu.
- Floor Selection: Allow users to select a floor from a dropdown menu.
- Date Selection: Enable users to choose a date from a calendar to view time slot availability.
- Time Slot Blocking: Enable admin users to block time slots for specific tables. User roles will be differentiated using access controls. 
- Time Slot Unblocking: Enable admin users to unblock previously blocked time slots. User roles will be differentiated using access controls.
- Availability Display: Display time slot availability for each table in a grid format.
- Table Summary: Generate a list of which tables are blocked, available, or booked on a given day, using reasoning to interpret available time slots to create a useful summary. Make the availability available via API endpoint.

## Style Guidelines:

- Primary color: Light purple (#A387D8) for a clean, modern feel.
- Background color: Light gray (#F2F2F7) to provide a soft, neutral backdrop.
- Accent color: Dark purple (#6B46C1) for interactive elements and buttons.
- Body and headline font: 'Inter' for a clean and readable UI.
- Use simple, outline-style icons for clarity and ease of understanding.
- Grid-based layout for the time slot display, with clear separation between slots and tables.
- Subtle transitions and animations when blocking or unblocking time slots.
